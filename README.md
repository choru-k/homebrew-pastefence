# Homebrew Tap for PasteFence

Official Homebrew tap for [PasteFence](https://github.com/choru-k/PasteFence) - a macOS menu bar app that masks sensitive information before pasting.

## Install

```bash
brew tap choru-k/pastefence
brew install --cask pastefence
```

## Update

```bash
brew upgrade --cask pastefence
```

## Uninstall

```bash
brew uninstall --cask pastefence
```
